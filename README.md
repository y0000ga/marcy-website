# 馬克信箱介紹網站 Dear Marcy

馬克信箱介紹網站是一個使用 React.js 與 Typescript等工具所打造的前端專案，不只可以查看馬克信箱相關資訊，使用者可從該網站得知目前主持人居住地的天氣，透過年份篩選差看馬克信箱發展歷史，閱覽各年度追思紀念會與歌曲的 Youtube 影片，體驗繪畫式心理測驗，並可透過 Google map 查看提供馬克信箱聽眾優惠的商家。

### [網站入口](https://y0000ga.github.io/marcy-website/)

## 功能與畫面
✅**首頁與導覽列**
1. 首頁的紅色跑馬燈顯示目前台北天氣。
2. 導覽列可前往各頁面，並可以透過點擊繁中與簡中字樣，分別呈現繁體中文與簡體中文網站內容。

<div align=center><kbd><img width="70%" src="https://i.imgur.com/ueyB9R1.png" alt="homePage"/></kbd></div>

✅**心理測驗**
1. 根據提供圖形進行聯想繪畫，並提出兩個相關的形容詞
2. 完成後可點擊按鈕以查看該題心理測驗解析

<div align=center><kbd><img width="70%" src="https://i.imgur.com/DQLbYRi.png" alt="psyTest" /></kbd></div>

✅**歷史閱覽**
1. 閱覽 2013 至今的馬克信箱發展歷史大事記
2. 可根據年度篩選特定時間的歷史事件

<div align=center><kbd><img width="70%" src="https://i.imgur.com/zce4tWZ.png" alt="history" /></kbd></div>

✅**名言閱覽**
1. 閱覽馬克信箱主持人的所有名言
2. 點擊獲得啟示按鈕，可獲得隨機的一則啟示
<div align=center><kbd><img width="70%" src="https://i.imgur.com/ni3kgUH.png" alt="saying" /></kbd></div>

✅**Youtube 影片閱覽**
1. 觀賞各年度的追思紀念會影片紀錄
2. 可獲得每一部影片目前的閱覽數、like 數與留言數資訊。
<div align=center><kbd><img width="70%" src="https://i.imgur.com/O1lrccX.png" alt="memory" /></kbd></div>

✅**優惠區**
1. 獲得提供聽眾專屬優惠的商家資訊。
2. 可透過類別篩選特定商家資訊。
3. 可從 Googlemap 上查看各個實體商家地點。

## 安裝
### 取得專案
```
git clone https://github.com/y0000ga/marcy-website.git
```
### 移動到專案內
```
cd marcy-website
```
### 安裝套件
```
npm install
```
### 運行專案
```
npm start
```
### 開啟專案
在瀏覽器網址列輸入以下即可看到畫面
```
http://localhost:3000/
```

## 專案主要技術
* React
* Typescript
* Tailwindcss
* React Router Dom
* React-hook-form
* @react-google-maps/api
* gsap
* react-i18next
